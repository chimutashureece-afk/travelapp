# Assets Directory

This directory contains static assets for the Olori Animate application.

## Structure

- `icons/` - Application icons for different platforms
- `images/` - UI images and graphics
- `templates/` - Template files for animations
- `sounds/` - Audio assets

## Icon Requirements

- `icon.png` - 512x512 PNG for Linux
- `icon.ico` - Windows icon file
- `icon.icns` - macOS icon file

## Usage

Assets in this directory are bundled with the application during the build process.
