# Exports Directory

This directory contains generated animations and exported files.

## Structure

- Generated animations are saved here with unique IDs
- Files are organized by date and generation ID
- Temporary files are cleaned up automatically

## File Naming Convention

Generated files follow this pattern:
`{generation_id}_{timestamp}.{extension}`

Example: `abc123_1703123456.mp4`

## Cleanup

Old files may be automatically cleaned up to save disk space.
Users can also manually manage files through the application interface.
